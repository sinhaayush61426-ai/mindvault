## Database Models & Data Flow

```
User
в”ңв”Җв”Җ username (unique)
в”ңв”Җв”Җ email (unique)
в”ңв”Җв”Җ password_hash
в””в”Җв”Җ relationships:
    в”ңв”Җв”Җ entries вҶ’ DiaryEntry (1:Many)
    в””в”Җв”Җ characters вҶ’ Character (1:Many)

DiaryEntry
в”ңв”Җв”Җ title
в”ңв”Җв”Җ category
в”ңв”Җв”Җ encrypted_content (stored as binary)
в”ңв”Җв”Җ timestamp
в”ңв”Җв”Җ release_date (optional for time-locked)
в”ңв”Җв”Җ user_id вҶ’ User (FK)
в””в”Җв”Җ relationships:
    в””в”Җв”Җ snapshots вҶ’ EntrySnapshot (1:Many)

Character
в”ңв”Җв”Җ name
в”ңв”Җв”Җ archetype
в”ңв”Җв”Җ personality_traits
в”ңв”Җв”Җ backstory
в”ңв”Җв”Җ relationships (JSON)
в”ңв”Җв”Җ physical_description
в”ңв”Җв”Җ motivations
в”ңв”Җв”Җ avatar_color
в”ңв”Җв”Җ user_id вҶ’ User (FK)
в””в”Җв”Җ timestamps (created_at, updated_at)

EntrySnapshot
в”ңв”Җв”Җ version
в”ңв”Җв”Җ encrypted_content
в”ңв”Җв”Җ timestamp
в”ңв”Җв”Җ description
в”ңв”Җв”Җ is_autosave (boolean)
```
